  ** Version 1.2 (일부 변경사항 반영되지 않음)

  * 몬스터 변경점

  - 보스 라운드에서 보스를 무시하고 일반 몬스터만 상대하는 플레이를 억제하기 위해, 보스가 베이스에 가하는 데미지를
    고정 20 -> 20 / 30 / 40으로 변경 (16R, 31R에서 상승)
  - 대신 보스 라운드에서 양쪽 모두를 놓쳤을 때의 피해가 너무 커지는 것을 방지하기 위해, 일반 몬스터가 베이스에 가하는 데미지를
    2 / 3 / 4 -> 2 / 2.5 / 3 으로 변경 (16R, 31R에서 상승)
  - 과하게 빠르거나 느린 몬스터 이동속도 조정 (기존 베이스에 도달하는 시간 몬스터 별 8~30초 -> 12~25초)
  - 일반 몬스터 방어력 조정 (기존 0%~60%, 최대 2배까지 증가 -> 15%~50%, 최대 2.4배까지 증가)
  - 보스 몬스터 방어력 조정 (기존 60% -> 70%), 체력 계수 20% 감소

  * 유닛 밸런스

  티어 보너스 변경
  - 1T : 해당 티어 +100%, 전체 +25% -> 해당 티어 +150%, 전체 +40%
  - 2T : 해당 티어 +20%, 전체 +10% -> 해당 티어 +40%, 전체 +20%
  - 3T : 해당 티어 +10%, 전체 +6% -> 해당 티어 +20%, 전체 +12%
  - 4T : 해당 티어 +5%, 전체 +2% -> 해당 티어 +8%, 전체 +3%

  화염족 대마도사 이프리트, 휘몰아치는 화염의 에단
  - 화상 데미지가 과도하게 적용되는 버그 수정
  > 툴팁으로는 공격력의 10%, 30%로 표기되었으나, 실제로는 110%, 130%로 적용되는 버그가 확인되어 수정함
  - 또한 중독과의 일관성을 유지하기 위해 틱 시간을 1초로 통일함
  - 따라서 이프리트, 에단의 패시브를 5초동안 1초마다 공격력의 40%, 120%로 변경

  힘의 부족 대족장 후씽크
  - 다른 1T 유닛들에 비해서 패시브의 메리트가 부족하다고 느껴짐
  - 공격력 500 -> 650
  - 3타마다 현재 체력 계수 8% -> 10%

  버서커 카이토닐
  - 가동률을 높인 기존 방식은 개성이 부족하다고 느껴지고, 다른 1티어 유닛에 비해서 약함
  - 따라서, 기본 공격력을 대폭 상향하고 공격속도를 소폭 하향해 패시브가 발동되었을 때 확실한 파워를 가지도록 변경
  - 패시브 지속시간 4초 -> 3초
  - 공격력 150 -> 360
  - 공격속도 1.4 -> 1.2

  되살아난 마검사 펠소닌
  - 기본 공격력이 너무 낮아 패시브를 제대로 활용하기 어려움
  - 공격력 200 -> 300

  타이사마
  - 낮은 공격속도에 대한 리턴으로 강력한 데미지를 내는 유닛이지만, 넓은 공격범위에 비해 너무 강력하다고 판단됨
  - 패시브 공격력 증가 배율 800배 -> 700배

  다크엘프 전투대장 히스
  - 패시브의 위력이 살짝 부족해서 상향
  - 패시브 계수 현재 체력의 2% -> 2.5%

  숲 관찰대장 세트
  - 툴팁 오류 수정 (3타마다 160% 데미지 -> 3타마다 공격력의 100% 추가 피해)

  고위 빙결 마도사 아케리스
  - 넓은 범위에 긴 지속시간으로 방어력을 감소시키는 패시브는 1명을 기용할 때는 효율이 과도하게 좋고, 여러 명을 기용할 때 효율이 떨어짐
  - 패시브 공격한 적에게 5초간 방어력 20% 감소시키는 디버프 부여 -> 공격한 적에게 3초간 이동속도를 8% 감소시키는 디버프 부여 (중첩 가능 - 곱연산)
  - 공격속도 0.7 -> 0.6

  암살대장 애퍼스
  - 초반에 강하고 후반에 약해지는 기획 의도에 맞게, 패시브를 조금 더 강력하게 변경
  - 패시브 공격력 증가 배율 8배 -> 10배

  미치광이 타이탄
  - 공격 범위가 짧은 유닛이라 패시브를 온전하게 활용하기 어려움
  - 패시브를 상향하고, 공격속도를 상향해 패시브를 조금 더 잘 활용하게 변경
  - 공격속도 0.9 -> 1.5
  - 공격력 230 -> 200
  - 패시브 잃은 체력 비레 최대 공격력 증가량 100% -> 200%

  마지막 후손 페르난도 (기존 3T 용기병 전사)
  - 아케리스의 패시브가 변경됨에 따라, 유닛을 2티어로 변경하고 방어력 감소 패시브 이동
  - 패시브 : 공격한 적에게 3초간 방어력을 15% 감소시키는 디버프 부여 (중첩 가능 - 곱연산)
